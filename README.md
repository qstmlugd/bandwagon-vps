## 拿到 VPS 后怎么连？SSH 登录完整步骤

买完不知道怎么用，这是新手最常卡的一步。VPS 就是一台装在机房里的电脑，你需要通过 SSH 远程登录它。

### 第一步：获取 IP 和端口

登录 KiwiVM 控制面板（邮件里有链接，或者在官网 Client Area 找到你的 VPS 点进去）。

面板首页能看到：
- **Public IP address**：服务器的公网 IP
- **SSH Port**：SSH 端口（搬瓦工的端口不是标准的 22，是一个随机五位数）
- root 密码需要在面板里重置一次才能用

### 第二步：重置 root 密码

在 KiwiVM 面板，找到 Root Password Modification，先把 VPS 关机，再点重置密码。密码只显示一次，复制下来存好。

### 第三步：用 SSH 工具连接

**Windows 用户：** 推荐用 Termius，免费、跨平台、界面干净。新建连接填入 IP、端口、用户名 root，密码粘贴进去，连接。

**macOS/Linux 用户：** 直接开终端，输入：

bash
ssh root@你的IP地址 -p 你的SSH端口


回车后输入密码（输密码时屏幕不会显示任何字符，这是正常的，盲打完回车就行）。

连接成功后你会看到命令行提示符，类似 `root@bandwagonhost:~#`，这就说明进来了。

### 第四步：重装系统（推荐）

搬瓦工默认安装的是 CentOS，现在更推荐换成 **Ubuntu 22.04 或 24.04**，社区资源更丰富，教程更多。

在 KiwiVM 面板找到 Install new OS，选 ubuntu-22.04-x86_64-bbr，勾选同意，点 Reload，等待几分钟即可。重装后系统是干净的，密码重新生成，重复上面的步骤重新登录。

## 机房怎么选？不同运营商答案不一样

买完之后最常问的问题：我选哪个机房？

搬瓦工目前有 20 多个机房，真正适合国内用户的高端机房主要是这几个，按运营商分开说：

**电信用户**

优先选 DC6（洛杉矶 CN2 GIA-E）或 DC9（洛杉矶 CN2 GIA）。双程走 CN2 GIA，延迟通常在 150-180ms，晚高峰也比较稳。DC9 机房是 AMD EPYC + NVMe SSD，性能比 DC6 更强一些。

**联通用户**

日本大阪软银（JPOS_1）表现不错，延迟低、丢包少、三网都能接受。荷兰联通 9929（EUNL_9）也是联通精品线路，如果你业务面向欧洲可以考虑。

**移动用户**

移动走 CMIN2 回程，DC6 和 DC9 机房都有优化，软银机房对移动也友好。

切换机房的操作很简单：KiwiVM 面板 → Migrate to another DC，选好目标机房，数据无损迁移，通常几分钟完成，不影响已有数据。

## 建站场景：WordPress 搭建基本流程

拿到 VPS 连上 SSH 之后，建站最常用的方式是装宝塔面板，比手动配环境省事很多。

**1. 安装宝塔面板**

Ubuntu 系统下运行：

bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh


安装完会显示面板地址、用户名和密码，用浏览器打开这个地址就能看到图形界面。

**2. 在宝塔里安装 LNMP 环境**

进宝塔面板 → 软件商店，找 LNMP 一键安装包，选 Nginx + MySQL 8.0 + PHP 8.1，点安装。等个十几分钟，环境就配好了。

**3. 新建网站**

宝塔面板 → 网站 → 添加站点，填入域名（域名需要提前解析到 VPS 的 IP），配置数据库，确认。

然后在网站目录上传 WordPress 安装包，按提示安装，几分钟搞定。

**套餐买完能升级吗？**

可以提工单申请套餐升级，差价补足即可。降级不支持。

**能部署 AI 应用吗？**

搬瓦工 VPS 没有 GPU，但可以调用外部 API 部署 Dify、n8n 这类 AI 应用。CN2 GIA-E 40G 及以上的套餐（2GB 内存起）跑 Docker Compose 比较顺，1GB 内存的套餐运行 Dify 会比较吃力。
